# Culinary_AI_Assistant<br>
1-	Create a Python 3.11.5 environment. <br>
2-	Install the requirements.txt file “pip install -r requirements.txt”.<br>
3-	Add your OpenAI API key to .env file.<br>
4-	In the Python environment, run the command “streamlit run streamlit_app.py” to start the streamlit application.<br>
